File Uploading
**************
File uploading dose not mean the uploading of the medias to database.
it is because the database like mysql, oracal are capable of uploading the data in
form of text format.
even though if you want to upload the data, (other then text such that) it has some medias
related mime type then we may go for, large text datatype, or BLOB object.

NOTE
*****
you can upload the image to the db where large amount of data is a constraint like mongodb,
S3(simple stage service) bucket(AWS).

In order to upload the data in the server using php
****************************************************
1- Make sure your php allowes file uploading feature php.ini check 
allow_file_uploading : on while uploading data you can upload maax to max 20 files
at once and each file size, can be 8MB post data but if you want you can Increase this limit you
can do it by gonig to php.ini and modifying the setting.

Follow three step for file uploding
************************************
1. $_FILE : 2D array
        1- name : name of the file
        2- size : file size
		3- error : any error accured during uploading not related file
		4- type : file type
		5- tmp_name : tmp location where filr is uploading
		
2. use function move_uploaded file($tmplocation, $newlocation); where
file is finally uploaded.

3. send the uploaded file name to the db.
